Project Stack

Backend

Framework & Server

  • FastAPI (Python) — REST API framework
  • Uvicorn — ASGI server

Database

  • SQLite via SQLAlchemy 2.0 ORM — stored at data/agentic_demo.db

AI / LLM

  • OpenAI SDK — direct API calls (GPT-4o, GPT-4, GPT-3.5, GPT-5, etc.)
  • LiteLLM proxy — optional routing layer for multi-model support (Anthropic, Ollama, etc.)
  • Lakera Guard — AI security/guardrail layer for prompt injection detection (backend/lakera.py)

RAG (Retrieval-Augmented Generation)

  • ChromaDB — vector store for embeddings (backend/rag.py)
  • PyPDF2 — PDF parsing for document ingestion

Agent / Tool System

  • Custom agent loop in backend/agent.py
  • MCP (Model Context Protocol) support via backend/mcp.py and backend/toolhive.py for tool discovery

Auth / Security

  • python-jose — JWT token handling
  • passlib[bcrypt] — password hashing

Frontend

Framework

  • React 18 + TypeScript
  • Vite 6 — build tool / dev server

Routing

  • React Router v6

Styling

  • Tailwind CSS v3
  • tailwind-merge + clsx — conditional class utilities

UI Libraries

  • lucide-react — icon library
  • react-dropzone — file upload UI

Summary: An AI demo / guardrail showcase app — a React + TypeScript SPA frontend talking to a FastAPI backend that routes LLM calls through OpenAI directly or via a LiteLLM proxy, enforces AI safety with Lakera Guard, supports RAG over uploaded documents via ChromaDB, and exposes agentic tool use via MCP.
